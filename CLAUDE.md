# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Обзор проекта

Интерактивные справочники по Пятикнижию Моисея на основании Синодального перевода Библии.

**Деплой:** https://dik-garri.github.io/Bible/

## Язык

Весь контент на русском языке. Ответы и комментарии в коде — по умолчанию на русском.

## Разработка

Нет сборки, линтеров или тестов. Статические HTML-файлы.

```bash
# Локальный сервер (нужен для fetch JSON-файлов)
python3 -m http.server 8000
# или
npx serve .
```

Деплой: GitHub Pages из ветки `main`. Пуш в main → автоматическое обновление сайта.

### JavaScript — ключевые функции

- `checkNavScroll()` (1487) — градиент-индикатор скролла навигации
- `toggleCard(header)` (1507) — раскрытие/сворачивание карточек
- `sortTable(colIndex)` (1514) — сортировка таблицы сравнения
- `BOOK_MAP` (1533–1573) — русское сокращение → ID файла (`'Лев'` → `'lv'`)
- `BOOK_FULL_NAME` (1575–1597) — ID файла → полное русское название
- `loadBook(fileId)` (1602) — загрузка JSON с кэшированием в `bookCache`
- `parseRef(text)` (1618) — парсинг ссылок типа `"Лев. 1:3–9"`, `"Быт. 22:2, 13–14"`
- `parseVerses(spec)` (1656) — парсинг диапазонов и перечислений стихов
- `openVerseModal(refText)` (1688) — показ модального окна с текстом стихов

### Кликабельные ссылки на Библию

Элементы `.ref`, `.cal-ref`, `.principle-verse` кликабельны. При клике парсится русское сокращение через `BOOK_MAP`, загружается `synodal/{fileId}.json`, стихи отображаются в `#verseOverlay`.

## Синодальный перевод (synodal/)

JSON-файлы: массив глав, каждая глава — массив строк стихов (0-indexed).

```javascript
// Левит 1:4
const data = await fetch('synodal/lv.json').then(r => r.json());
const verse = data[0][3]; // глава 1 → индекс 0, стих 4 → индекс 3
```

Основные файлы Пятикнижия: `gn.json` (Бытие), `ex.json` (Исход), `lv.json` (Левит), `nm.json` (Числа), `dt.json` (Второзаконие).
