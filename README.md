# Пятикнижие Моисея

Интерактивные справочники по темам Пятикнижия Моисея на основании текста Синодального перевода.

**Демо:** [dik-garri.github.io/Bible](https://dik-garri.github.io/Bible/)

## Темы

### Жертвоприношения (`sacrifices.html`)

- **5 основных жертв** (Левит 1–7) — раскрывающиеся карточки с цветовым кодированием
- **7 особых жертв** — посвящение священников, Йом Киппур, рыжая телица, Пасха, назорей, завет, патриархи
- **Календарь жертвоприношений** — от ежедневных до праздника Кущей (Числа 28–29)
- **Сравнительная таблица** — сортируемая таблица 5 основных жертв
- **Распределение жертвы** — визуальные диаграммы: Богу / священнику / приносящему
- **8 богословских принципов** жертвенной системы

### Чистота и нечистота (`purity.html`)

- **Пища** — чистые и нечистые животные (Левит 11, Второзаконие 14)
- **Тело** — роды, истечения, поллюция, менструация (Левит 12, 15)
- **Проказа** — диагностика, очищение, проказа на одежде и доме (Левит 13–14)
- **Мёртвое тело** — осквернение трупом, рыжая телица (Числа 19)
- **Стан** — святость стана, военная чистота, проказа Мариам (Числа 5, Второзаконие 23)
- **Сводная таблица** — сортируемая таблица всех видов нечистоты
- **8 богословских принципов** системы чистоты

### Праздники и священные собрания (`feasts.html`)

- **Еженедельные** — суббота и новомесячия
- **Весенние** — Пасха и Опресноки, Первые снопы, Пятидесятница (Левит 23, Исход 12)
- **Осенние** — Праздник Труб, Йом Киппур, Кущи с таблицей убывающих тельцов (Числа 29)
- **Годовые циклы** — субботний год, юбилейный год, три паломничества (Левит 25)
- **Сводная таблица** — сортируемая таблица всех 11 праздников
- **8 богословских принципов** праздничной системы

### Скиния (`tabernacle.html`)

- **Общий план** — приношения народа (15 материалов), конструкция (брусья, покровы), мастера Веселеил и Аголиав
- **Двор** — жертвенник всесожжений, умывальник, ограда (100×50 локтей)
- **Святое** — стол хлебов предложения, золотой светильник (менора), жертвенник курения
- **Святое Святых** — ковчег завета с херувимами, завеса, слава Господня (Исход 40)
- **Облачения** — ефод, наперсник с 12 камнями, верхняя риза с колокольчиками, кидар
- **Служение левитов** — каафиты, гирсониты, мерариты (Числа 3–4)
- **8 богословских принципов** устройства скинии

### Заветы (`covenants.html`)

- **Ноев** — благословение, закон о крови, вечный завет, радуга (Бытие 9)
- **Авраамов** — призвание, завет рассечения (Бытие 15), обрезание (Бытие 17)
- **Синайский** — богоявление, Десять заповедей, Книга завета, ратификация кровью, обновление (Исход 19–24, 34)
- **Моавитский** — благословения, проклятия, обрезание сердца, призыв к жизни (Второзаконие 28–30)
- **Сравнительная таблица** — сортируемая таблица 4 заветов
- **8 богословских принципов** заветной системы

### Общие возможности

- **Подтягивание текста Библии** — клик на любую ссылку (Лев. 1:4, Быт. 22:2 и т.д.) открывает модальное окно с текстом стихов из Синодального перевода

## Формат данных

Каждый JSON-файл в `synodal/` — массив глав, каждая глава — массив строк (стихов), с нулевой индексацией:

```javascript
// Иоанна 3:16
const data = await fetch('synodal/jo.json').then(r => r.json());
const verse = data[2][15]; // глава 3 (индекс 2), стих 16 (индекс 15)
```

## Технологии

Статические HTML-файлы с общим CSS (`css/shared.css`) и JavaScript (`js/shared.js`). Без фреймворков, без сборки, без зависимостей. Тексты Библии загружаются по клику через `fetch()` из JSON-файлов.
