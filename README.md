# Жертвы в Пятикнижии Моисея

Интерактивный справочник по всем видам жертвоприношений в Пятикнижии Моисея на основании текста Синодального перевода.

**Демо:** [dik-garri.github.io/Bible](https://dik-garri.github.io/Bible/)

## Возможности

- **5 основных жертв** (Левит 1-7) — раскрывающиеся карточки с цветовым кодированием, описанием ритуала, допустимых животных и ключевых стихов
- **7 особых жертв** — посвящение священников, День Очищения, рыжая телица, Пасха, назорейские жертвы, жертва завета, патриархальные жертвы
- **Календарь жертвоприношений** — от ежедневных до праздника Кущей (Числа 28-29)
- **Сравнительная таблица** — сортируемая таблица 5 основных жертв
- **Распределение жертвы** — визуальные диаграммы: кому что достаётся (Богу, священнику, приносящему)
- **8 богословских принципов** жертвенной системы
- **Подтягивание текста Библии** — клик на любую ссылку (Лев. 1:4, Быт. 22:2 и т.д.) открывает модальное окно с текстом стихов из Синодального перевода

## Структура

```
Bible/
├── index.html                         # Интерактивная страница (один файл, без зависимостей)
├── Жертвы_в_Пятикнижии_анализ.md      # Исходный анализ в Markdown
└── synodal/                           # Русский Синодальный перевод (JSON)
    ├── gn.json                        # Бытие
    ├── ex.json                        # Исход
    ├── lv.json                        # Левит
    ├── nm.json                        # Числа
    ├── dt.json                        # Второзаконие
    └── ...                            # Все 66 книг Библии
```

## Формат данных

Каждый JSON-файл в `synodal/` — массив глав, каждая глава — массив строк (стихов), с нулевой индексацией:

```javascript
// Иоанна 3:16
const data = await fetch('synodal/jo.json').then(r => r.json());
const verse = data[2][15]; // глава 3 (индекс 2), стих 16 (индекс 15)
```

## Технологии

Один HTML-файл со встроенными CSS и JavaScript. Без фреймворков, без сборки, без зависимостей. Тексты Библии загружаются по клику через `fetch()` из JSON-файлов.
